=================================================================
Mini Calculatrice - Python 
=================================================================


Description :
---------------
Ce projet est une mini calculatrice en ligne de commande écrite en Python. 
Elle permet d'effectuer des opérations de base (addition, soustraction, multiplication, division) entre deux nombres entrés par l'utilisateur.


 Fonctionnalités :
--------------------
- Menu interactif avec choix des opérations
- Saisie de deux nombres (décimaux acceptés)
- Vérification des erreurs de saisie
- Protection contre la division par zéro
- Exécution en boucle jusqu’à ce que l’utilisateur quitte


 Fichier principal :
----------------------
- `calculatrice.py` : contient le code source du programme


Prérequis :
--------------
- Python 3.x installé
- Terminal ou éditeur de texte (VS Code, PyCharm, IDLE, etc.)


Lancement :
--------------
1. Cloner le dépôt ou télécharger le fichier `calculatrice.py`
2. Exécuter dans un terminal :python calculatrice.py


Utilisation :
------------------
1. Lancer le programme
2. Sélectionner une opération en entrant un chiffre de 1 à 5 :
   - 1 : Addition
   - 2 : Soustraction
   - 3 : Multiplication
   - 4 : Division
   - 5 : Quitter
3. Entrer les deux nombres demandés
4. Lire le résultat affiché
5. Refaire une opération ou quitter


Exemple :
------------
--- Calculatrice ---
1. Addition
2. Soustraction
3. Multiplication
4. Division
5. Quitter
Choisissez une opération (1-5) : 2
Entrez le premier nombre : 12
Entrez le deuxième nombre : 5
Résultat : 7.0


Sécurité :
-------------
- Si un nombre non valide est entré : message d'erreur
- Si division par zéro : message d'erreur
